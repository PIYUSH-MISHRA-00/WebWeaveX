# Getting Started

WebWeaveX is under active development. The repository currently provides the production-grade structure, governance, and documentation scaffolding required for a serious open-source platform. Implementation of the core crawler engine begins in Phase 1.

## Prerequisites

- Git
- Python 3.11+
- Node.js 18+ (for Playwright and website tooling)

## Repository Setup

```bash
git clone https://github.com/PIYUSH-MISHRA-00/WebWeaveX
cd WebWeaveX
```
