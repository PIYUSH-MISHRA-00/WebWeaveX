# WebWeaveX v0.1.0 Release Notes

Release date: 2026-03-18

## Highlights

- Production-ready SDKs for Python, Node.js, Dart, Java, and Kotlin.
- Configurable retries, backoff, timeout controls, and structured errors across all SDKs.
- Typed Java/Kotlin models for stronger compile-time guarantees.
- Improved CI coverage across all supported language ecosystems.
- Upgraded documentation and onboarding experience for faster developer adoption.

## Included Work

- Core validation: CLI + API endpoints validated against `https://example.com`.
- SDK validation: all language examples executed against a local API server.
- Packaging validation:
  - Python wheel/sdist build + local install/import.
  - Node tarball build + global install verification.
  - Dart publish dry-run validation.
  - Java/Kotlin Maven install validation.
- Website validation: Docusaurus production build success.

## Upgrade Notes

- Existing Java/Kotlin integrations should migrate from raw maps/strings to typed models.
- Python/Node/Dart SDK consumers can optionally enable debug logs for transport-level troubleshooting.

## Security Notes

- Use HTTPS endpoints for production deployments.
- Validate user-supplied target URLs before invoking crawl operations.
- Apply outbound network policy restrictions for crawling workers.
