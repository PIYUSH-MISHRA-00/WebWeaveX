# SDKs

WebWeaveX provides SDKs for Python, Node.js, Java, Kotlin, and Dart. All SDKs expose the same high-level API to ensure consistent behavior across ecosystems.

## Design Goals

- Identical method signatures across languages
- Streaming-friendly responses for large crawls
- Typed models and clear error handling
- Stable, versioned contracts

## Planned Core APIs

- `crawl(url)`
- `rag(url)`
- `research(topic)`
- `knowledge_graph(url)`
