# Security Policy

## Responsible Disclosure

We take security seriously. If you discover a vulnerability, please report it privately and do not open a public issue.

Report steps:
- Email security@webweavex.dev with a clear description, impact, and reproduction steps.
- If possible, include a minimal proof-of-concept.
- We will acknowledge receipt within 72 hours and work with you on a fix and disclosure timeline.

Please do not disclose the issue publicly until we have released a fix and coordinated a disclosure date.

## Supported Versions

During the 0.x development series, only the `main` branch and the latest tagged release are supported with security fixes. Older tags are not supported.

## Security Contact

Primary contact: security@webweavex.dev
