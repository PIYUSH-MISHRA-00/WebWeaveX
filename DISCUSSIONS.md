# GitHub Discussions

We use GitHub Discussions for community collaboration and longer-form conversations. Please choose the most appropriate category:

- Announcements: major releases, project milestones, and maintainer updates.
- Q&A: usage questions and troubleshooting help.
- Ideas: proposals for new features or improvements.
- RFCs: design discussions that may influence architecture or public APIs.
- Show and Tell: share what you built with WebWeaveX.

Before starting a new discussion, search existing threads to avoid duplicates and include as much context as possible.
