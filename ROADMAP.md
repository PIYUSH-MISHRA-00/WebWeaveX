# Roadmap

This roadmap outlines major milestones and themes for WebWeaveX. Timelines will be refined as the project evolves.

## WebWeaveX v1.0

- Core crawler engine
- SDKs (Python, Node.js, Java, Kotlin, Dart)
- CLI
- Plugin system

## WebWeaveX v2.0

- Distributed crawler
- Autonomous research engine
- Knowledge graph

## WebWeaveX v3.0

- Global crawl network
- AI-guided crawling
- Self-learning extraction models
